This are all the contents of SafeHome's openHAB configuration ( /conf ) folder. 
To ensure your downloaded version of SafeHome works as intended, please copy the contents of this folders (specifically, items, rules, sitemaps, things and transform)
to your instance of the openHAB server you wish to run SafeHome on to.

** IMPORTANT **
Please also copy the .jar file provided in this folder (org.openhab.binding.safehome_se.[...].jar) to your instance of the openHAB server at the path /addons/ .
For example, if your openHAB version is the release 2.5.2, the path where you will need to copy and paste this file would be openhab-2.5.2/addons/ .
